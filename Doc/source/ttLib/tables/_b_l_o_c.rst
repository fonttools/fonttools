``bloc``: Bitmap Location table
-------------------------------

The ``bloc`` table is an Apple Advanced Typography (AAT) table.

Its structure is identical to that of the ``EBLC`` table. It is paired with 
the ``bdat`` table, which has the same structure as the ``EBDT`` table.
