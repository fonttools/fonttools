``bhed``: Font Header table
----------------------------

The ``bhed`` table is an Apple Advanced Typography (AAT) table.

Its structure is identical to that of the ``head`` table. OS X uses the 
presence of a 'bhed' table as a flag that a font doesn't have any glyph outlines 
but only embedded bitmaps.
