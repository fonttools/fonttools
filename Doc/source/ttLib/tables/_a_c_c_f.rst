``accf``: Apple Color Compressed Font table
-------------------------------------------

The ``accf`` table is a private Apple table used in colour-emoji fonts
on iOS 8.3 through iOS 9.3. It stores glyph images using a proprietary
run-length and palette-indexed compression scheme. Two wire format versions
exist: version 1 (magic ``\x40\x30\x20\x10``) and version 2 (magic ``fcca``).
