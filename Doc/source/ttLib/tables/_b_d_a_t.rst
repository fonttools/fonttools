``bdat``: Bitmap Data table
---------------------------

The ``bdat`` table is an Apple Advanced Typography (AAT) table.

Its structure is identical to that of the ``EBDT`` table. It is paired with 
the ``bloc`` table, which has the same structure as the ``EBLC`` table.
