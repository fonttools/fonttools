``IFTX``: Incremental Font Transfer Extension table
----------------------------------------------------

The ``IFTX`` table is part of the Incremental Font Transfer specification.
It provides extension data for the ``IFT`` table.

See also: https://w3c.github.io/IFT/Overview.html
