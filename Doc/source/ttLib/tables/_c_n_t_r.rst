``cntr``: Apple Emoji Centering Data table
------------------------------------------

The ``cntr`` table is an Apple proprietary table used in colour-emoji fonts
(e.g., AppleColorEmoji). Its payload is a UTF-8 encoded JSON object describing
per-glyph bounding boxes and centering offsets, keyed by glyph ID.
